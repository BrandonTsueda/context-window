# Context Window MVP

## Product Summary

Context Window helps a user understand messy project history by extracting structure from unstructured text.

## Primary User Story

As a user, I can paste a conversation or document and quickly see:

- a short summary
- key decisions
- action items
- unresolved questions

## MVP Screens

1. Home / landing page
2. New entry page
3. Project view
4. Entry detail view

## Core Entities

- Project
- Source entry
- Summary
- Decision
- Action item
- Open question
